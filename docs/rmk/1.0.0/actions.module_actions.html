Function of converts object keys to snake_case.

Example
let formula = rmk(toSnakeCase())
let data = {fooBar:1, foo_bar:2, bar:3, Foo:4}
formula(data)
// => {foo_bar:1, foo_bar:2, bar:3, foo:4}
